[golang]如何正确关闭一个channel

如何优雅地关闭channel

close一个channel的原则

  • 永远只在发送端close一个channel，也就是不要在接收端close一个channel
  • 不要直接close一个有多个发送者的channel，除非能确定负责关闭的是最后一个还在活动的发送者

如果需要处理上面的情况，可以考虑使用sync.once或者sync.mutex设置一个指示channel已关闭的变量，或者添加一个safeSend 函数，代码如下

func safeSend(ch chan<- T, value T) (closed bool) {
    defer func() {
        // 如果channel已经close，发送会导致panic，内置函数recover不会返回nil
        if recover() != nil {
            closed = true
        }
    }()
    ch <- value
    closed = false
    return
}

这种方式的缺点是不能使用内置的select的语句

有两种更优雅的解决方式

  1. 需要从1个接收端关闭channel的情况

设置1个用来发送信号的channel，用来通知所有发送端不要再发送数据给channel了

```go package main

import ( "log" "math/rand" "sync" "time" )

func main() { const maxRandomNum = 10000 const sendersNum = 100 done := make(chan struct{}) dataCh := make(chan int, 100)

log.SetFlags(0)
rand.Seed(time.Now().UnixNano())

wgSenders := sync.WaitGroup{}

for i := 0; i < sendersNum; i++ {
    wgSenders.Add(1)
    go func() {
        defer wgSenders.Done()
        for {
            select {
            case <-done:
                return
            case dataCh <- rand.Intn(maxRandomNum):
            }
        }
    }()
}
for v := range dataCh {
    log.Println(v)
    if v == maxRandomNum-1 {
        close(done)
        break
    }
}
wgSenders.Wait()
log.Println("done")
close(dataCh)

} ```

  1. 需要从多个接收端或发送端关闭channel的情况 设置一个仲裁者，当需要关闭channel时，通知仲裁者关闭channel

```go package main

import ( "log" "math/rand" "sync" "time" )

func main() { const maxRandomNum = 10000 const sendersNum = 100 const receiversNum = 10 done := make(chan struct{}) toDone := make(chan struct{}) dataCh := make(chan int, 100)

log.SetFlags(0)
rand.Seed(time.Now().UnixNano())

wgSenders := sync.WaitGroup{}
wgReceivers := sync.WaitGroup{}

for i := 0; i < sendersNum; i++ {
    wgSenders.Add(1)
    go func() {
        defer wgSenders.Done()
        for {
            v := rand.Intn(maxRandomNum)
            if v == 0 {
                select {
                case toDone <- struct{}{}:
                default:
                }
                return
            }
            select {
            case <-done:
                return
            case dataCh <- v:
            }
        }
    }()
}

for i := 0; i < receiversNum; i++ {
    wgReceivers.Add(1)
    go func() {
        defer wgReceivers.Done()
        for {
            select {
            case <-done:
                return
            case v := <-dataCh:
                log.Println(v)
                if v == maxRandomNum-1 {
                    select {
                    case toDone <- struct{}{}:
                    default:
                    }
                    return
                }
            }
        }
    }()
}

<-toDone
close(done)
wgSenders.Wait()
wgReceivers.Wait()
log.Println("done")
close(dataCh)

} ```
