排序算法总结

排序算法总结

冒泡排序

伪代码

function bubble_sort(array,length)
    for i from 0 to length-1
        for j from 0 to length-1-i
            if array[j] > array[j+1]
                swap array[j] and array[j+1]

选择排序

伪代码

function select_sort(array,length)
    for i from 0 to length-1
        currentmini is array[i]
        for j from i+1 to length-1
            if array[j] < currentmini
                currentmini is array[j]
        swap array[i] and currentmini

插入排序

类似抓扑克牌

伪代码

function insert_sort(array)
    array[0] is sorted
    for each unsorted element x
        for j from last_sorted_index to 0
            if array[j] < x
                move element j+1 to last_sorted_index right by 1
                insert x in array[j+1]
                break loop

堆排序

堆是一种特殊的二叉树,大堆表示父结点元素均大于子结点的堆,小堆相反

结点序号从0开始,则有如下性质:

parent(i) = floor((i-1)/2)
leftchild(i) = 2i+1
rightchild(i) = 2(i+1)

伪代码

function heap_sort(array,length)
    function max_helpify(root,end)
        loop
            if max of child < root or > end
                break loop
            swap root and max of child
            index of root = index of max of child

    # build max heap
    for node from last no-leaf node to root
        max_helpify(node,length-1)

    # heap sort
    for node from last node to root
        swap array[0] and node
        max_helpify(array[0],index of node)

快速排序

思想

选择一个基准(pivot),将小于该基准的元素放在它左边,大于的放右边,然后以该元素为界将数组分为两部分,重复次过程直到数组长度小于1,则此时该数组已排序完毕
